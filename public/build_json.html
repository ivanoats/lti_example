Data App Builder

Apps that are made up completely of collections of resources can be built as data apps. This page provides a utility for quickly building or for better understanding the JSON data format used by data apps.

Introduction

Some LTI apps require lots of custom functionality in order to work, but some apps can be handled as simply a collection of links that can be inserted into course content. If your app, like a bunch of app on this site, delivers a collection of links to rich content, then you can quickly and easily implement it as an LTI app by building a JSON data set of the collection of resources. You can use the table builder below to manually create resources, then click "Generate JSON". Or you can paste in JSON you've built and make sure it parses correctly by clicking "Parse JSON".

Once you've built your data set, upload it to a public server that allows CORS and submit the URL as a "Data App" (or just submit the actual JSON if there are fewer than 500 records) on the Suggestions page.
